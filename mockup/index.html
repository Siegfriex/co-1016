주요 기능

Phase 1: 현재 가치 분석

레이더 차트와 선버스트 차트로 아티스트의 현재 가치를 시각화합니다.

Phase 2: 커리어 궤적 분석

시계열 데이터로 아티스트의 커리어 발전 과정을 추적합니다.

Phase 3: 비교 분석

두 아티스트를 비교하여 유사점과 차이점을 분석합니다.

Phase 4: AI 보고서

AI가 생성한 종합 분석 보고서를 확인합니다.
