# 🎨 CuratorOdyssey Phase 1 + AI Analysis

> 데이터로 가치의 지도를 그리고, AI가 통찰의 길을 제시합니다

## 🚀 빠른 시작

```bash
# 의존성 설치
npm install

# 개발 서버 시작
npm start

# 브라우저에서 http://localhost:3000 열기
```

## 🎯 Phase 1 주요 기능

### ✨ 핵심 시각화
- **레이더 차트**: 5축 가치 요약 (I/F/A/M/Sedu)
- **선버스트 차트**: 4축 근거 데이터 (제도/학술/담론/네트워크) 
- **AI 인사이트 패널**: GPT-4 기반 전문가 수준 분석 🤖
- **인터랙티브 연동**: 레이더 축 호버 → 선버스트 하이라이트

### 🎨 디자인 시스템
- **DYSS 기반**: 0930masterdesign_1.md의 통합 디자인 시스템 적용
- **다크모드 지원**: 사용자 시스템 설정 자동 감지
- **완전 반응형**: 모바일부터 데스크톱까지 최적화
- **접근성**: WCAG 2.1 AA 준수

### 🔧 기술 스택
- **React 18** + **D3.js 7**: 고성능 데이터 시각화
- **OpenAI GPT-4** + **Anthropic Claude**: AI 분석 엔진
- **Firebase**: 실시간 데이터베이스 연동 준비
- **CSS Custom Properties**: 디자인 시스템 토큰화
- **Jest + RTL**: 종합 테스트 스위트 (27개 테스트 케이스)

## 🤖 AI 분석 기능

### **자동 인사이트 생성**
- Phase 1 진입 시 자동으로 GPT-4 분석 실행
- 가치구성 특징, 균형도, 시장포지셔닝, 발전제언 4섹션 구조
- 30분 캐시로 비용 절약 및 성능 최적화

### **다중 백업 시스템**
- **1차**: OpenAI GPT-4 (메인 분석 엔진)
- **2차**: Anthropic Claude (백업 AI)
- **3차**: 통계 기반 폴백 분석 (AI 실패 시)

### **지능형 에러 처리**
- 일일 한도 초과, 네트워크 오류 등 구체적 안내
- 자동 재시도 (최대 3회) 및 우아한 실패 처리
- 사용자 친화적 오류 메시지

## 🤖 AI 기능 설정

**1단계**: 프로젝트 루트에 `.env.local` 파일 생성:
```bash
REACT_APP_OPENAI_API_KEY=your_openai_api_key_here
REACT_APP_OPENAI_MODEL=gpt-4
```

**2단계**: [OpenAI Platform](https://platform.openai.com/)에서 API 키 발급

**3단계**: 앱 실행 시 자동으로 AI 분석 활성화!

📖 **자세한 설정 가이드**: [AI_SETUP_GUIDE.md](./AI_SETUP_GUIDE.md)

## 🧪 테스트 범위

- **AI Service**: OpenAI 연동, 폴백 처리, 에러 핸들링 (15개 테스트)
- **AI Cache**: LRU 캐싱, TTL 관리, 메모리 최적화 (12개 테스트)
- **Integration**: React 컴포넌트 연동 테스트

## ⚠️ 보안 주의사항

- **개발 환경**: `dangerouslyAllowBrowser` 플래그 사용 중
- **프로덕션**: 반드시 백엔드 API로 이전 필요
- **API 키**: `.env.local` 파일이 Git에 커밋되지 않도록 주의
- **비용 모니터링**: OpenAI 사용량 정기적 확인 권장
